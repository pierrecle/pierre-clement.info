Jenkins exécute les scripts que j'ai développés, qui exécutent les tests unitaires, fait la build, créé un tag Git pour figer la version de l'application si besoin, et la livre automatiquement sur Elastic Beanstalk avec le bon numéro de version. Donc en gros, pour livrer, il suffit de pousser sur la branche correspondante à l'environnement souhaité.
